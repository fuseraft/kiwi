# console

### `clear()`

Clears the console.

### `read(msg = "")`

Read console input.

**Parameters**
| Type | Name | Description |
| :--- | :--- | :--- |
| `String` | `msg` | Optional. A message to prompt the end-user. |

### `write(msg = "")`

Writes console output.

**Parameters**
| Type | Name | Description |
| :--- | :--- | :--- |
| `String` | `msg` | Optional. A message to print. |
