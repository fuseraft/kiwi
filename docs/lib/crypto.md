# `crypto`

The `crypto` package contains functionality for generating hash strings.

---

## Package Functions

### `md5_hash(input = "")`
Generates an MD5 hash string.

**Parameters**

| Type | Name | Description |
| :--- | :--- | :--- |
| `string` | `input` | The input string.|

**Returns**

| Type | Description |
| :--- | :--- |
| `string` | An MD5 hash. |
