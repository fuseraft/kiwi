- [`collections`](lib/collections.md) ‚Äî Specialized collection types, including `Heap` and `Set`.
- [`compress`](lib/compress.md) ‚Äî Functions for compression and decompression.
- [`console`](lib/console.md) ‚Äî An interface that wraps core I/O operations.
- [`crypto`](lib/crypto.md) ‚Äî Cryptographic functions like MD5 and SHA-2.
- [`csv`](lib/csv.md) ‚Äî Functions for parsing CSV data.
- [`env`](lib/env.md) ‚Äî For interacting with environment variables.
- [`fio`](lib/fio.md) ‚Äî Functions for file system operations.
- [`http`](lib/http.md) ‚Äî Functions for HTTP/HTTPS requests.
- [`iter`](lib/iter.md) ‚Äî An explicit iterator pattern for safely traversing lists.
- [`math`](lib/math.md) ‚Äî Common mathematical functions and utilities.
- [`path`](lib/path.md) ‚Äî Contains the definition for the `Path` struct.
- [`reflector`](lib/reflector.md) ‚Äî Reflection tools, including `Callable`.
- [`socket` / `tls`](lib/socket.md) ‚Äî Low-level asynchronous TCP and TLS networking.
- [`stdin`](lib/stdin.md) ‚Äî Functions for reading from standard input.
- [`string`](lib/string.md) ‚Äî String manipulation and transformation utilities.
- [`sys`](lib/sys.md) ‚Äî For executing shell commands.
- [`task`](lib/task.md) ‚Äî Asynchronous task management, including `Channel`.
- [`tester`](lib/tester.md) ‚Äî A simple unit testing framework.
- [`time`](lib/time.md) ‚Äî Time and date utilities.
